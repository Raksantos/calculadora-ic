# Calculadora IC

Ferramenta adotada para auxílio de alunos dos cursos de computação do **Instituto de Computação (IC) — UFAL**.

[🔗 Link do site hospedado](https://raksantos.github.io/calculadora-ic/)

## 📌 Sobre o projeto

A Calculadora do IC nasceu em **2019**, como projeto de três alunos da então disciplina de *Internet e Web*, do curso de Ciências da Computação da UFAL. De lá pra cá o mundo mudou bastante (pandemia, ensino remoto, novas turmas), mas o propósito segue o mesmo: **feito por alunos do IC para a comunidade do IC e da UFAL**.

Saiba mais na página [Sobre](./sobre.html) do site.

## 🧮 Funcionalidades

- **Faltas** — calcule quantas faltas você pode ter sem reprovar por frequência.
- **Nota Final** — descubra qual nota você precisa tirar na final para ser aprovado.
- **Almoço** — some quanto vai gastar com almoço no R.U.
- **Café** — some quanto vai gastar com café no R.U.
- **Passagens** — calcule o gasto com passagens de ônibus.

## 💻 Rodando localmente

O projeto é estático (HTML + CSS + jQuery/Bootstrap via CDN), então não precisa de build. Escolha uma das formas:

**Abrir direto no navegador**
```bash
xdg-open index.html   # Linux
open index.html       # macOS
start index.html      # Windows
```

**VS Code** — instale a extensão *Live Server*, clique com o direito em `index.html` → *Open with Live Server*.

## 🤝 Contribuindo

Os autores originais já não têm tanto tempo para manter o projeto ativamente, mas o **repositório é público e aberto a Pull Requests**. Algumas formas de contribuir:

- Abra uma **issue** descrevendo bugs ou sugestões.
- Envie uma **Pull Request** com correções, melhorias ou novas funcionalidades.
- Caso queira ajudar a revisar issues/PRs de forma recorrente, pode ser adicionado como **moderador** do projeto.
- Sinta-se à vontade para criar um **fork** e publicar uma versão melhorada — o objetivo é sempre melhorar e ajudar o IC.

### Aprender e praticar, antes de tudo

Estamos em um mundo bem diferente do de 2019 e **encorajamos o uso de IA** no dia-a-dia do desenvolvimento. Ainda assim, o intuito do projeto também é ser um **ambiente onde alunos possam colocar em prática suas ideias e novas features**. A ideia aqui, antes de tudo, é **aprender e praticar** — pegar um repositório real, abrir uma PR, discutir código, errar, corrigir e crescer no processo.
